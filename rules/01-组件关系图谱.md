# 组件关系图谱

本页按「页面生成可调用性」整理组件。组件 `description` 与 `cover` 当前均为空，以下关系主要来自组件名称、props、slots 和变量命名；推断项已单独标注。

## A. 页面生成优先入口

这些组件可以作为 D2C 生成界面的主要调用对象。

| 场景 | 优先组件 | 关键变体 |
| --- | --- | --- |
| 皮肤、壁纸、字体、社区等封面商品卡 | `封面卡片组件` | `属性 1`: 壁纸 / 字体 / 皮肤 / 社区；`属性 2`: 价格 / 常规 / 描述 / 多行 |
| 图标类分类卡 | `图标卡片组件` | `属性 1`: checked / unchecked |
| 可点选卡片 | `点击卡片组件` | 不带品类_未选中 / 不带品类_选中 / 带品类_未选中 / 带品类_选中 |
| 上传入口卡片 | `上传卡片组件` | 上传 / 回显 / 选项 |
| 键盘展示 | `键盘组件` | 按键皮肤键盘 1 / 按键皮肤键盘 2 |
| 开关、单选、复选列表项 | `list_switch` | theme: radio / checkbox / toggle；status: unchecked / checked；info: with/without description；icon: with/without icon |
| 进入型列表项 | `list_enter` | normal / enter_withmark / enter_withdescription / enter_withpic / enter_withdescription_withdescripition / enter |
| 弹窗 | `modal` | Btn-WithContent / 1Btn-NoContent / 2Btn-WithContent / 2Btn-WithOptions |
| Toast | `toast` | max / min |

## B. 基础控件

这些组件可以直接用于页面，但通常承担局部交互或文本展示。

| 类型 | 组件 | 用法 |
| --- | --- | --- |
| 按钮 | `button` | 主按钮 / 次按钮；可点击 / 不可点击 / 加载 / 按下 / 失败 / 已启用；sm / md / lg |
| 特殊按钮 | `ButtonSpecial` | AccentButton / FocusButton / VipButton / textbutton |
| 按钮组 | `button_group` | confirm / group |
| 图标 | `icon` | back / file / close / voice / enter |
| 图标按钮 | `iconButton` | 无 props，谨慎使用 |
| 单选 | `radio` | checked / unchecked |
| 复选 | `checkbox` | unchecked / checked |
| 开关 | `toggle` | on / off |
| 输入框 | `input` | single / multi |
| 抽屉输入 | `drawer_input` | multi / single；slot: `input` |
| 抽屉头部 | `drawer_header` | primary / secondary；with description / with icon / with icontext / without description |
| 选项 | `option` | centered / left-aligned；withIcon / withoutIcon；selected / unselected |
| 标题 | `title` | title / title_description. |
| 段落 | `paragraph` | 无 props |
| 占位态 | `placeholder` | 容器 36 / 容器 37 |
| 类别标签 | `类别标签` | 皮肤背景 / 皮肤按键 / 字体 / 社区等 |
| 玩法标签 | `玩法标签（素材营销、账号权益）` | 推荐 / 热门 / 趣玩组件 / 限免等 |

## C. 复合组件的内部结构

这些组件更像复合卡片内部的组成部件。D2C 生成完整页面时，不应优先直接调用它们，除非目标只是还原该局部。

| 组件 | 关系 |
| --- | --- |
| `卡片_封面` | 推断为 `封面卡片组件` 的封面区域，变体：壁纸 / 常规 / 字体 |
| `卡片_内容区` | 推断为 `封面卡片组件` 的文本、标签、价格信息区域 |
| `图标卡片组件-图标` | 推断为 `图标卡片组件` 内部图标状态，变体：动效 / 字体 / 按键 / 皮肤 / 背景 / 贴纸，状态：选中 / 未选中 |
| `withicon` | 推断为列表项内部的图标+文本块，通常服务于 `list_switch` 的 `icon=withicon` |

## D. 禁止作为页面组件入口的素材

以下组件是图片资源素材，只应作为视觉资源理解，不应作为 D2C 页面生成的业务组件入口。注意：`玩法标签（素材营销、账号权益）` 虽然名称包含“素材营销”，但它是标签组件，不属于本禁用范围。

| 素材组件 | 禁用原因 |
| --- | --- |
| `1:1 图片素材` | 图片资源组，不承载业务结构 |
| `3 纵列图片素材` | 图片资源组，不承载业务结构 |

## E. 说明类或不明组件

| 组件 | 策略 |
| --- | --- |
| `使用说明 title` | 规范说明组件，不用于业务页面 |
| `组件 3` | 仅有 alipay / wepay 变体，推断为支付渠道图形或标识；没有明确页面角色前不要主动调用 |
| `placeholder_pic` | 占位态图片素材；优先通过 `placeholder` 表达完整占位态 |
