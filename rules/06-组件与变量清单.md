## 图标事实

`icons.json` 当前为：

```json
{ "icons": [] }
```

因此不能使用 `<ui-icon>` 调用库图标。需要图标时优先使用 `icon` 组件已有变体；仍不能表达时再按 MasterGo 页面生成规则使用 FontAwesome，并在提交前说明原因。
