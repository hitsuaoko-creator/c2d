## 使用规则

1. 生成页面时优先使用变量名，不直接抄变量值。
2. 浅色界面使用 `浅色_index_1/*`，深色界面使用 `深色_index_1/*`，不要混用。
3. 如果浅色变量有而深色变量缺失，例如浅色 `图片组背底色`，深色模式不得臆造对应变量。
4. 当前库没有字号、间距、圆角、阴影变量；这些属性允许按页面协议使用字面值，但要遵守 8pt 间距和组件原始视觉。

## 引用示例

```html
<div
  data-name="page"
  class="bg-[var(浅色_index_1/15-整体背景色)] text-[var(浅色_index_1/1 - 一级文本色)]"
></div>
```

```html
<div
  data-name="card"
  class="bg-[var(浅色_index_1/10 - 一级卡片色)] border-[var(浅色_index_1/19-描边色)]"
></div>
```
