# 安装与适配

`c2d` supports Codex, Cursor, and Claude Code slash-style use.

## Codex

Install to:

```bash
~/.codex/skills/c2d
```

Invoke from the skill list or send:

```text
/c2d
c2d
设计
```

## Cursor

Install the skill to:

```bash
~/.cursor/skills/c2d
```

For project-level slash commands, install:

```bash
<project>/.cursor/commands/c2d.md
```

## Claude Code

Install the skill to:

```bash
~/.claude/skills/c2d
```

For command-list slash invocation, install:

```bash
~/.claude/commands/c2d.md
```

or project-level:

```bash
<project>/.claude/commands/c2d.md
```

## Recommended Script

Use:

```bash
scripts/install_c2d.sh all
```

Project-level command install:

```bash
scripts/install_slash_commands.sh /absolute/project both
```
