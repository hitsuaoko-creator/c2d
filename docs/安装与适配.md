# 安装与适配

`c2d` 支持 Codex、Cursor、Claude Code 三端使用，并尽量提供接近 `/c2d` 的斜杠唤起体验。

## Codex

安装到：

```bash
~/.codex/skills/c2d
```

可以从 Skill 列表唤起，也可以直接发送：

```text
/c2d
c2d
设计
```

## Cursor

安装 Skill 到：

```bash
~/.cursor/skills/c2d
```

如需项目级斜杠命令，安装到：

```bash
<project>/.cursor/commands/c2d.md
```

## Claude Code

安装 Skill 到：

```bash
~/.claude/skills/c2d
```

如需命令列表中的斜杠唤起，安装到：

```bash
~/.claude/commands/c2d.md
```

或安装项目级命令到：

```bash
<project>/.claude/commands/c2d.md
```

## 推荐脚本

一次安装三端：

```bash
scripts/install_c2d.sh all
```

安装项目级命令：

```bash
scripts/install_slash_commands.sh /absolute/project both
```
