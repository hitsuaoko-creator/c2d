# 常见触发语

These should trigger c2d:

```text
/c2d
c2d
设计
使用 c2d
$c2d
输入法APP设计规范
基于交互截图生成输入法APP设计稿
基于 MasterGo 组件库生成高保真 UI
```

If `/c2d` is invoked with no arguments, start guided intake immediately.

Ask only for missing blockers:

1. 目标 MasterGo 画布链接
2. 交互截图或截图路径
3. 组件库是否使用默认链接
4. 是否已有 visual-insight 产物
